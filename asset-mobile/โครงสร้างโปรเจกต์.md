# โครงสร้างโปรเจกต์ Mobile App

## โครงสร้างไฟล์

```
asset-mobile/
├── App.js                      # Main App Component และ Navigation
├── app.json                    # Expo Configuration
├── babel.config.js             # Babel Configuration
├── package.json                # Dependencies
├── README.md                   # คู่มือหลัก
├── SETUP.md                    # คู่มือการตั้งค่า API URL
├── QUICK_START.md             # คู่มือเริ่มต้นใช้งาน
│
└── src/
    ├── components/             # Components ที่ใช้ร่วมกัน
    │   └── common/
    │       └── Loading.jsx     # Loading Component
    │
    ├── contexts/               # React Contexts
    │   └── AuthContext.jsx     # Authentication Context
    │
    ├── hooks/                  # Custom Hooks
    │   └── useAuth.js          # Authentication Hook
    │
    ├── screens/                # หน้าจอต่างๆ
    │   ├── LoginScreen.jsx     # หน้าล็อกอิน
    │   ├── DashboardScreen.jsx # หน้า Dashboard
    │   ├── ScanScreen.jsx      # หน้าสแกน QR Code
    │   ├── AssetsScreen.jsx    # หน้ารายการครุภัณฑ์
    │   ├── CheckScreen.jsx     # หน้าตรวจสอบครุภัณฑ์
    │   ├── BorrowsScreen.jsx   # หน้าการยืม-คืน
    │   └── ProfileScreen.jsx   # หน้าโปรไฟล์
    │
    ├── services/               # API Services
    │   ├── api.js              # Axios API Client
    │   └── auth.js             # Authentication Service
    │
    └── utils/                  # Utilities
        └── constants.js        # Constants และ Config
```

## หน้าจอหลัก

### 1. LoginScreen
- หน้าล็อกอิน
- ใช้ credentials เดียวกับเว็บไซต์

### 2. DashboardScreen
- แสดงสถิติครุภัณฑ์
- เมนูด่วนเข้าถึงฟีเจอร์ต่างๆ
- Pull to refresh

### 3. ScanScreen
- สแกน QR Code ด้วยกล้อง
- กรอกรหัสครุภัณฑ์ด้วยมือ
- แสดงข้อมูลครุภัณฑ์
- บันทึกการตรวจสอบ

### 4. AssetsScreen
- รายการครุภัณฑ์ทั้งหมด
- ค้นหาครุภัณฑ์
- Pull to refresh
- ดูรายละเอียดครุภัณฑ์

### 5. CheckScreen
- รายการครุภัณฑ์ที่ยังไม่ตรวจสอบ
- ไปหน้าสแกนเพื่อตรวจสอบ

### 6. BorrowsScreen
- ประวัติการยืม-คืนครุภัณฑ์
- แสดงสถานะการยืม

### 7. ProfileScreen
- ข้อมูลผู้ใช้
- ออกจากระบบ

## Navigation Structure

```
App
├── AuthProvider
    └── NavigationContainer
        ├── LoginScreen (if not authenticated)
        └── MainTabs (if authenticated)
            ├── Dashboard
            ├── Scan
            ├── Assets
            ├── Check
            ├── Borrows
            └── Profile
```

## Dependencies หลัก

- **expo**: Framework สำหรับ React Native
- **@react-navigation**: Navigation library
- **axios**: HTTP client สำหรับ API calls
- **@react-native-async-storage/async-storage**: สำหรับเก็บ token และ user data
- **expo-barcode-scanner**: สำหรับสแกน QR Code
- **@expo/vector-icons**: Icons library

## API Integration

แอปใช้ API เดียวกับเว็บไซต์:
- Base URL: กำหนดใน `src/utils/constants.js`
- Authentication: Bearer Token (เก็บใน AsyncStorage)
- Endpoints: ใช้ endpoints เดียวกับเว็บไซต์
